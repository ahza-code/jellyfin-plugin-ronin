# Ronin

An Anime toolkit for Jellyfin designed to enhance library management and metadata visualization.

## Features

* **Global Anime Re-Org**: Automatically restructure your anime library into Aired Seasons (TVDB order) or flatten everything into a single season.
* **Filler & Canon Tracking**: Automatically identifies and tags episodes as filler, mixed, or canon based on online databases.
* **Visual UI Badges**: Adds clear visual labels to episode lists, making it easy to identify filler content at a glance.
* **Precise Control**: Supports tag-based filtering to target specific series for reorganization tasks.
